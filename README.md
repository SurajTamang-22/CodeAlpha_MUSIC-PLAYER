📌 Project Overview

Web Music Player is a responsive, web-based application that allows users to upload, manage, and play music seamlessly. It provides essential music controls, playlist management, and search functionality, delivering a smooth and user-friendly listening experience.

🚀 Features

✅ Upload & Play Music — Supports local music files using HTML5 audio
🎶 Playlist Management — Create and manage custom playlists
⏯️ Music Controls — Play, Pause, Skip, and Volume control
🔍 Search & Categorization — Search songs by name, artist, or genre
🌗 Dark/Light Mode (Optional) — Clean UI with theme toggle support
📱 Responsive Design — Works smoothly on desktop and mobile devices

🛠️ **Tech Stack**

Frontend:
HTML5 — Structure
CSS3 — Styling & responsiveness
JavaScript (ES6) — Logic & interactivity

Audio Handling:
HTML5 `<audio>` API with JavaScript controls


🧠 Key Concepts Used

• DOM Manipulation
• JavaScript Media APIs
• State Management (current track, playlist, playback status)
